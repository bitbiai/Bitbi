# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Bitbi, please do not report it publicly through Issues, Discussions, or Pull Requests.

Instead, please report it privately by email so the issue can be reviewed and fixed responsibly.

Contact:
- bit@bitbi.ai

Please include:
- a clear description of the issue
- steps to reproduce it
- the affected page, feature, or endpoint
- screenshots or proof of concept if helpful

I will review valid reports as soon as possible and work on an appropriate fix.
